This image allows you to run acme-dns without a config file with only a few environment parameters.

## Enviromnet variables
- **ACME_DNS_DOMAIN**
- **ACME_DNS_EMAIL**
- **ACME_DNS_IP** (optional)
- **ACME_DNS_NS** (optional)
- **ACME_DNS_RECORDS** (optional)
